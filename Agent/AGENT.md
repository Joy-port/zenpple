## 4｜Logo 使用規則

Logo 本身是墨色筆觸（書法字），需根據背景做混合模式處理：

| 背景類型 | 混合模式 | 效果說明 |
|----------|----------|----------|
| 白底、淺底、奶霧白 | 墨色直接使用（預設） | 乾淨有重量 |
| 深色底、彩色底 | Screen（濾色） | 深色消失，書法筆觸浮現為光感 |
| 奶霧白、淺彩底 | Multiply（正片疊底） | 墨色帶上底色溫度，logo 不孤立 |

**CSS 實作：**
```css
/* 深色底 */
.logo-on-dark { mix-blend-mode: screen; filter: invert(1); }
/* 彩色底課程卡 */
.logo-on-color { mix-blend-mode: screen; filter: invert(1); }
/* 淺底 */
.logo-on-light { mix-blend-mode: multiply; }
```

## 5｜視覺設計規則

### 整體氣質
- 現代禪意、乾淨留白、有重量感
- 不神秘主義包裝，真實感優先
- 書法筆觸 + 現代排版 = 品牌核心美學張力

### 底色場景規則
| 場景 | 底色 | 說明 |
|------|------|------|
| 淺底（主要） | `#F2EFEA` 奶霧白 | 書法圖 / 墨跡紋理 → Multiply |
| 深色底 | `#141414` 墨 | 書法圖 → Screen + invert，只留筆觸光痕 |
| 彩色底（課程卡） | 各頁主題色 | 頌缽波動紋理 → Screen，浮在色域上層 |

### 課程卡視覺（有機色域卡片）
三大課程系列各有漸層卡片色調：
- **QI-SB** — 青藍綠漸層（`#4A6B8A` 系）
- **HL** — 粉紫漸層（`#C47B7B` + `#7B6B9E`）
- **SC** — 霧藍暮紫漸層（`#7B6B9E` 深紫系）

卡片結構：課程代碼（Mono）→ 課程名稱（ZH Bold）→ 英文名稱（Josefin 100）→ 簡介（Body）→ 底部標注（Space Mono）

### Action Button 風格
- 實心黑底按鈕（`#2A2A2A`）+ 白字
- 無圓角或極小圓角
- hover 狀態：反色或墨色加深

### 大膽排版版本（Bold Composition）
- 字型當視覺元素用：超大、出血、不對稱
- 技法：出血大字、對角分割、垂直大字塔、極簡反差、情緒出血字
- 前提：用色和材質仍走品牌氣息，先撞到人、再留住人

## 11｜視覺參考圖庫解讀（Mood Board）

此區段記錄客戶提供的視覺參考圖，並說明每張圖對 ZENPPLE 網站設計的啟發方向。Agent 在生成視覺元素、背景處理、排版策略時應對照這些感受描述。

---

### 11-A｜質感語言 — 水墨 × 有機形狀

**參考：** 中國瓷器彩釉海報（第 1 頁）、清明水彩插畫（第 2 頁）、中式水墨山水（第 21 頁）

**感受摘要：**
- 顏色是半透明的、有呼吸感的，不是飽和的實色
- 形狀邊緣「不乾淨」，有水跡、暈染、滲透——這是設計語言的一部分，不是缺陷
- 留白是主角，圖形只是留白裡漂浮的東西
- 東方美學不靠繁複堆疊，靠「一個東西放對位置」

**對 ZENPPLE 的應用指引：**
- 背景紋理可用半透明色塊、漸層暈染，避免實色填滿
- 裝飾元素（書法、波紋、有機形狀）要有「浮在頁面上」的感覺
- 課程卡的漸層不宜太銳利，要像顏料滲入紙張的柔邊
- CSS 實作：`filter: blur()` + 低透明度漸層 + `mix-blend-mode` 製造東方暈染感

---

### 11-B｜排版勇氣 — 字型即視覺元素

**參考：** Tomotake 展覽海報（第 3 頁）、法國劇場系列海報（第 5、6 頁）、David Shrigley 文字海報（第 20 頁）、CHINA 2024 海報（第 22 頁）

**感受摘要：**
- 字型不只是「說明」，本身就是畫面的主視覺
- 字間距、字級大小的極端對比創造張力（超大 vs 超小）
- 手寫字 / 有機字型和幾何型字體的混搭不突兀，反而有對話感
- 文字可以放歪、出血、不對齊——刻意打破網格反而更有力
- 法國劇場系列：每張海報只靠「一個有機色塊 + 手寫字」就撐起整體，說明極簡可以極有力

**對 ZENPPLE 的應用指引：**
- Hero 區塊的主標題（如「往內定頻」）可以做超大字，字本身就是視覺焦點
- Josefin Sans 100 的極細字體可以壓在圖像上方當半透明圖層，不搶主視覺
- 課程標題的中英文字級落差要刻意拉大（ZH 36px vs EN 12px），不要做成等大
- 頁面節奏可用：超大字節 → 大留白 → 小文字，製造呼吸感

---

### 11-C｜有機形狀系統

**參考：** Tropical Deli Café 圖形系統（第 4 頁）、Théâtre du Nord 劇場系列（第 6 頁）

**感受摘要：**
- 有機形狀（blob、不規則圓、手繪邊緣）比幾何形更有溫度
- 一個品牌可以用「同一個形狀語言」貫穿所有版面，形成系統感
- Tropical 的做法：同一組圖形庫，換顏色、換比例，就能產生無限變化
- 劇場海報系列：每張只換一個色塊形狀，整個系列就有識別感又不單調

**對 ZENPPLE 的應用指引：**
- ZENPPLE 的「有機形狀系統」建議以以下元素為核心：書法字的筆觸輪廓、頌缽的橢圓弧形、波紋的流動線條
- 這些形狀可重複出現在：課程卡背景、section 分隔裝飾、hover 狀態底色
- CSS 實作：用 `border-radius` 的有機數值（如 `63% 37% 54% 46% / 55% 48% 52% 45%`）製造 blob 感
- 每個課程系列的形狀語言可略有差異，但都從「流動、圓潤、不尖銳」出發

---

### 11-D｜噪點 × 顆粒質感

**參考：** Formes Vives 藝術家駐留海報（第 11 頁）、BAUM 品牌插圖（第 7 頁）、DNA2725 CHINA 海報（第 22 頁）

**感受摘要：**
- 噪點（grain）讓畫面從「螢幕感」變成「印刷感 / 手工感」
- 顆粒質感讓顏色有了深度，避免數位顏色的塑膠光滑
- 不需要大量噪點，只要一層低不透明度的 grain overlay 就夠
- BAUM 插圖的模糊邊緣、顆粒感，讓花朵圖像有「記憶中的顏色」的感覺

**對 ZENPPLE 的應用指引：**
- 全站建議加一層 grain texture overlay（SVG noise filter 或 CSS grain）
- 特別是深色底的區塊（Hero、dark section），grain 可以讓背景不死板
- CSS 實作範例：
```css
.grain-overlay::after {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url("data:image/svg+xml,..."); /* SVG noise */
  opacity: 0.035;
  pointer-events: none;
  mix-blend-mode: overlay;
}
```
- 漸層色塊加 grain 後會產生近似印刷 risograph 的效果，和品牌書法質感呼應

---

### 11-E｜模糊光暈 × 能量感視覺

**參考：** 粉紫能量花朵光場（第 13 頁）、紫藍漸層植物（第 17 頁）、模糊有機形日本展覽海報（第 16 頁）、彩釉器物方格圖（第 18 頁）

**感受摘要：**
- 高斯模糊（Gaussian blur）加漸層色 = 「能量場」的視覺語言
- 這種美學在靈性品牌中有強烈的直覺辨識度，但 ZENPPLE 的版本要更克制、更現代
- 發光感不靠白色，靠「比背景亮的同色系」——比如紫底上的淡粉光
- 彩釉器物方格圖的配色（粉、薰衣草、青、黃綠）和 ZENPPLE 色系高度接近

**對 ZENPPLE 的應用指引：**
- 課程卡的漸層背景可使用 `radial-gradient` 製造從中心向外散的光暈感
- SC（薩滿）系列最適合這種能量場視覺，可以加更多模糊光暈
- HL（深層對齊）Hero 可用暗底 + 微弱光暈，製造「能量正在流動」的感覺
- 避免：光暈過亮、顏色過飽和、過多光源——保持「含蓄的光」
- CSS 實作：`radial-gradient(ellipse at 40% 60%, rgba(200,150,180,0.4) 0%, transparent 60%)`

### 11-F｜書封 × 乾淨線條插圖

**參考：** Emma Andievska 詩集封面（第 10 頁）、"We Moved On" 插圖（第 23 頁）、"Have a good dream" 鳥插圖（第 8 頁）

**感受摘要：**
- 單色線條插圖（只用一種顏色）有強烈的品味感
- 人物、動物的形狀做得「夠扁平、夠乾淨」，反而比寫實畫更有記憶點
- "We Moved On" 的構圖：大量留白 + 分散在頁面各處的小人物 + 幾條斜線 = 整體感覺是詩、不是設計
- Emma Andievska 的植物線稿：細線 + 白底，圖形本身就是裝飾

**對 ZENPPLE 的應用指引：**
- 首頁三張角色卡的插圖風格，應參考這類「細線條 + 單色 + 有機形」的方向
- 插圖可以：細線輪廓（stroke only，no fill）/ 或單色平塗（flat fill，no gradient）
- 適合的題材：水草、山石、雲霧、波紋、空曠風景——無人形，但有「人在其中的感覺」
- 線條粗細可以有變化，但整體保持輕盈

### 11-H｜整體視覺方向總結（給 Agent 的判斷準則）

在生成任何 ZENPPLE 視覺元素時，請用以下三個問題自我檢查：

1. **有沒有呼吸空間？** — 頁面應該感覺「夠安靜」，能讓人放慢。密集排列是錯的。
2. **顏色有沒有「霧感」？** — 避免全飽和色。所有顏色都應該像是加了一層薄薄的白霧或水氣。
3. **有沒有手工感 / 有機感？** — 任何幾何圖形、排版結構都應該有一點「不那麼完美」的邊緣，像是書法、印刷、手繪的溫度。

如果三個都是 Yes，視覺方向就對了。
